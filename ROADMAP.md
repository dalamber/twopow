# Roadmap

## Shipped

- [x] Base-2 exponentiation with documented overflow.
- [x] Cross-platform CI matrix (OS × compiler × configuration).
- [x] Conan, vcpkg, and container (GHCR) distribution.
- [x] In-tree bindings: C API, Python, and WebAssembly.
- [x] Publishable SDK wrappers in seven languages (Java, Kotlin, Swift, C#, Go,
      Node.js, Ruby).
- [x] Exponentiation-as-a-Service (HTTP + OpenAPI + Swagger UI).
- [x] Fuzzing, sanitizer matrix, and static analysis.
- [x] Formal proof (`PROOF.md`), RFC-2119 specification (`SPEC.md`), and i18n docs.
- [x] ASCII logo and brand guidelines.

## Under evaluation

- [ ] Base-3 support (significant API and scope implications; under evaluation).
- [ ] Community Discord (coming soon).
- [ ] A graphical (non-ASCII) logo.

These items require resources and direction not yet committed.
