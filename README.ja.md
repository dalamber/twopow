# twopow

[English](README.md) · [Español](README.es.md) · [Deutsch](README.de.md) · [中文](README.zh-CN.md) · **日本語**

> モダン C++ のための、決定的で高性能な 2 を底とするべき乗ライブラリ。

`twopow` は厳密に仕様化された単一の操作を提供します。すなわち、呼び出し側が指定した
指数で **2** をべき乗します。予測可能な動作、網羅的なテストカバレッジ、そして一級の
クロスプラットフォーム対応を備えています。

## 特徴

- **決定的。** 同じ入力は、対応するすべてのプラットフォームで同じ出力を生成します。
- **明確に定義された定義域。** 有効な指数は閉区間 `[0, 63]` です。範囲外の指数は
  `twopow::overflow_error` を送出します。
- **実行時依存ゼロ。** C++20 標準ライブラリのみに依存します。
- **クロスプラットフォーム。** Linux・macOS・Windows 上で、GCC・Clang・MSVC により
  継続的に検証されています。

## クイックスタート

```cpp
#include <twopow/twopow.hpp>
#include <iostream>

int main() {
    std::cout << twopow::two_pow(10) << "\n";  // 1024
}
```
